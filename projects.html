1. Child Malnutrition Detection System Using CNN

This project was developed as a part of the final year project to address child malnutrition through automated detection using image analysis. The system uses a Convolutional Neural Network (CNN) to classify the nutritional status of children based on images. By analyzing visual features, the model can identify whether a child is Normal, Malnourished, or Over Malnourished.

A dataset of 4,000 images was collected from various sources, including Roboflow and other publicly available child malnutrition datasets. Data augmentation techniques such as rotation, zoom, and flipping were applied to improve the model’s accuracy and robustness.

The model was implemented using Python and TensorFlow/Keras, with OpenCV used for image preprocessing. The user interface (UI) was developed using Tkinter, allowing users to upload a child’s photo and receive a nutritional status prediction along with a confidence score.

Why This Project Matters

  • Many communities don’t have easy access to proper health monitoring tools.
  • Malnutrition can be hard to detect early.
  • A simple image-based tool can support faster screening and help health workers take action sooner.

How It Works

  1. Input Image: A photo of the child is uploaded.
  2. Preprocessing: The image is resized, normalized, and cleaned for model input.
  3. CNN Model: Extracts features from the face/body structure and classifies into categories: Normal, Malnourished, or Over Malnourished.
  4. Output: Displays prediction with confidence level through the Tkinter-based interface.

Technologies Used

  • Python
  • TensorFlow / Keras
  • CNN architecture (custom CNN)
  • OpenCV for image preprocessing
  • Tkinter for user interface

Dataset

  • 4,000 images collected from Roboflow and other publicly available datasets.
  • Data augmentation applied (rotation, zoom, flipping) to increase dataset diversity.

Key Features

  • Early detection using images
  • User-friendly interface with Tkinter
  • Lightweight model suitable for desktop deployment
  • Accurate predictions after training

Possible Use Cases

  • Rural health checkups
  • NGOs and health workers
  • Mobile health apps
  • Community nutrition programs

Impact

  • Supports preventive healthcare and child well-being.
  • Assists healthcare workers and NGOs in making data-driven decisions.
  • Promotes awareness about child nutrition in communities.

This college project was a dynamic web development assignment where we built an e-commerce website named "Oven House". The project was implemented using XAMPP as a local server, PHP for backend functionality, and MySQL for database management.

The website allows users to browse products, add items to the cart, and place orders. It includes user registration/login, product management, and dynamic content rendering from the database. The project demonstrates how dynamic websites can interact with data to provide an interactive shopping experience.

Why This Project Matters

3. IoT Smart Mirror

This project involved building a smart mirror prototype integrating IoT devices to create an interactive smart home experience. The mirror displays real-time weather updates, calendar events, and notifications directly on the mirror's surface.

The hardware consists of a display behind a two-way mirror, sensors, and a microcontroller (e.g., Raspberry Pi or Arduino) connected to the home network. Software integration fetches data from APIs and updates the mirror interface dynamically.

Why This Project Matters

  • Introduces IoT integration with consumer devices.
  • Demonstrates real-time data display and interactive interfaces.
  • Combines software and hardware knowledge for smart home solutions.

How It Works

  1. Display: Two-way mirror with embedded monitor shows data.
  2. IoT Integration: Connected to sensors and APIs for live updates.
  3. Data Display: Shows weather, calendar events, and notifications.
  4. Interactive Features: Supports touch or gesture controls (optional).

Technologies Used

  • Raspberry Pi / Arduino
  • Python / Node.js
  • APIs for weather and calendar
  • Display and sensors for smart mirror

Key Features

  • Real-time information on mirror surface
  • IoT-enabled smart home integration
  • Customizable dashboard
  • Interactive user experience
